# wicked-sales-js
A full stack Node.js and React shopping cart app.

## Live Demo

Try the application live at TBD

## Features
- User can view the products for sale
- User can view details of a product
- User can add a product to their cart
- User can add a product to their cart
- User can view their cart summary
- User can place an order

## System Requirements

- Node.js 10 or higher
- NPM 6 or higher
- PostgreSQL 6 or higher

#### Getting Started

1. Clone the repository.

```shell
git clone https://github.com/judyichiang/wicked-sales-js.git
```

2. Install all dependencies with NPM.
```shell
npm install
```

3. Start the project.

```shell
npm run dev
```

4. Once started, you can view the application by opening [https://localhost:3000](https://localhost:3000)
